## 12. 常见问题排查

### 12.1 前端打开 XML NoSuchKey

常见原因：

```text
dist 文件夹整体上传到了 OSS，而不是上传 dist 内部文件。
OSS 根目录没有 index.html。
OSS 静态网站没有配置默认首页 index.html。
CDN 还缓存旧错误。
```

修复：

```text
OSS 根目录直接放 index.html 和 assets/
配置默认首页 index.html
配置默认 404 index.html
刷新 CDN
```

### 12.2 注册返回 500

先看日志：

```bash
pm2 logs getoffer-api --lines 100 --nostream
```

可能原因：

```text
验证码 SceneId 前后端不一致。
验证码 AK/SK 缺失或权限不足。
验证码地域错误。
.env.production 未加载。
SQLite 路径权限问题。
```

必要时在：

```text
server/src/middleware/error.ts
```

临时加入：

```ts
console.error("[server-error]", error);
```

然后重新构建部署。

### 12.3 跨域错误

检查：

```env
FRONTEND_ORIGIN=https://www.getofferai.cn
```

重启：

```bash
NODE_ENV=production pm2 reload getoffer-api --update-env
```

### 12.4 前端请求 localhost

检查：

```text
client/.env.production
```

必须：

```env
VITE_API_BASE_URL=https://api.getofferai.cn
VITE_USE_MOCK=false
```

重新构建并上传 OSS。

### 12.5 API 502

检查：

```bash
pm2 status
pm2 logs getoffer-api --lines 100 --nostream
curl http://127.0.0.1:3001/api/health
sudo nginx -t
sudo systemctl status nginx
```

### 12.6 ASR 失败

检查：

```text
ALIYUN_AK_ID
ALIYUN_AK_SECRET
NLS_APP_KEY
ALIYUN_ASR_REGION
业务 OSS Bucket 权限
interview_audio/ 临时文件上传和删除
```

### 12.7 简历下载失败

检查 Agent 返回的 `docx_oss_key` 必须形如：

```text
generated_resumes/{safe_user_id}/{resume_id}.docx
```

后端会拒绝非当前用户路径。

如果使用 Git：

```bash

git log --oneline
git checkout <last-good-commit>
npm ci
npm run build
NODE_ENV=production pm2 reload getoffer-api --update-env

```

```text
用上一版 server 目录覆盖当前 server
```
